Using this 2016 House and Senate campaign finance data, can you find three candidates and build a spreadsheet so that it is more useful, so that you can ask specific questions of it?

Data Modeling in Excel, Part 2

Pick three sex offenders from the District's registry. How would you model their information in Excel? You'll want to narrow it down to common types of information about them and you probably won't want to model everything.

Be prepared to discuss in class what information you'd model (and wouldn't model) for both Excel files and your reasoning.
